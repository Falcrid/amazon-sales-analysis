# Amazon Sales Dataset Analysis

## Project Overview
Exploratory Data Analysis and Machine Learning project on 1,400+ Amazon products to uncover pricing patterns and predict product ratings.

## Technologies Used
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- Machine Learning (Scikit-learn)

## Key Findings
- Analyzed 1,465 products across 10+ categories
- Average discount rate: 50%+ across all products
- Rating count is the strongest predictor of product rating
- Random Forest model outperformed Linear Regression

## Machine Learning
- **Models:** Linear Regression, Random Forest
- **Target:** Product Rating prediction
- **Best Model:** Random Forest (RÂ² = 0.107, MAE = 0.187)
